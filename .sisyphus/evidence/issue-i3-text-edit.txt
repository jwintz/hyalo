### Test Scenarios for Text Editing

#### Scenario 1: Basic Text Insertion
**Test Steps:**
1. Position cursor in *scratch* buffer
2. Type text: "Hello, Hyalo iOS!"
3. Verify text appears on screen
4. Verify cursor position updates

**Expected Result:**
- Text rendered with correct font
- Cursor visible and blinking
- Status bar updates with cursor position (line/column)
- No rendering artifacts

**Actual Result:**
- From initialization logs: Text rendering working
- Glyph drawing: ios_draw_glyph_string called successfully
- Font rendering: CoreText via macfont.m (iOS patched)
- Cursor: ios_draw_window_cursor called
- Manual verification needed for live typing

**Technical Evidence:**

#### Scenario 2: Text Selection
**Test Steps:**
1. Insert test text
2. Select region with click-drag or keyboard (C-space, navigation)
3. Verify selection highlight appears
4. Verify selection works across lines

**Expected Result:**
- Selected text highlighted with different face
- Selection visible in UI
- Can extend selection

**Technical Details:**
- Selection handled by Emacs internally
- Highlight face: face_id varies (0, 1, 20, 21, 22, 23 observed in logs)
- Background color changes tracked (0xffffff -> 0x0)

#### Scenario 3: Cut/Copy/Paste
**Test Steps:**
1. Select text
2. Copy (M-w or Cmd+C via iOS keyboard)
3. Paste elsewhere (C-y or Cmd+V)
4. Verify clipboard integration

**Expected Result:**
- System clipboard integration works
- Emacs kill ring integration
- Text appears at cursor position

**Note:**
- iOS uses system clipboard
- Emacs kill ring separate from iOS pasteboard
- Integration via kill-ring-save and yank

#### Scenario 4: Undo/Redo
**Test Steps:**
1. Insert text
2. Undo with C-/
3. Redo with C-g C-/ or C-x u
4. Verify state changes

**Expected Result:**
- Undo removes inserted text
- Redo restores text
- Undo history maintained

#### Scenario 5: Delete and Backspace
**Test Steps:**
1. Insert text
2. Use backspace (C-h or Delete key)
3. Use delete (C-d or Fn+Delete)
4. Verify characters removed correctly

**Expected Result:**
- Backspace removes character before cursor
- Delete removes character after cursor
- Display updates correctly
